# chmod-x

Collection of practical bash scripts for everyday productivity.

## Scripts

### go - Quick Link Manager
Lightning-fast bookmark system for the terminal.
```bash
go yt              # Open YouTube
go add tw https://twitter.com
go list            # Show all links
go remove tw
```

## Installation
```bash
git clone https://github.com/yourusername/chmod-x.git
cd chmod-x
./install.sh
```
